## Store requirements

- **iOS App Store:** 1024×1024 px PNG, no transparency (or with transparency if desired).
- **Android Play Store:** Same icon is used; adaptive icon uses `adaptive-icon.png` (foreground).

## Setup

1. **Download the logo** (e.g. open the URL above in a browser and Save Image As).
2. **Resize to 1024×1024** (recommended) in an image editor and export as PNG.
3. **Replace these files** in this folder:
   - `icon.png` – main app icon (1024×1024 recommended)
   - `adaptive-icon.png` – Android adaptive icon foreground (1024×1024)
   - `splash-icon.png` – splash screen image
   - `favicon.png` – web favicon (e.g. 48×48)
